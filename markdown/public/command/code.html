13.2. 缩进式多行代码

缩进 4 个空格或是 1 个制表符

一个代码区块会一直持续到没有缩进的那一行（或是文件结尾）。

代码：

    #include <stdio.h>
    int main(void)
    {
        printf("Hello world\n");
    }

显示效果：

#include <stdio.h>
int main(void)
{
    printf("Hello world\n");
}

13.4. 代码高亮

代码高亮示例:

/**
* nth element in the fibonacci series.
* @param n >= 0
* @return the nth element, >= 0.
*/
function fib(n) {
  var a = 1, b = 1;
  var tmp;
  while (--n >= 0) {
    tmp = a;
    a += b;
    b = tmp;
  }
  return a;
}

document.write(fib(10));
